
# Comment Checker

An AI-powered comment moderation app built with **Streamlit** and **OpenAI**.  
This tool helps detect **offensive, harsh, vulgar, or rule-breaking comments** before they are posted — perfect for blogs, forums, and community dashboards.

##  Features
- Real-time comment moderation using OpenAI’s Moderation API
- Detects offensive, hate, sexual, or violent language
- Simple, clean Streamlit UI
- Secure API key management via Streamlit Secrets
- “Submit” and “Clear” actions for smooth workflow

##  Tech Stack
- **Python**
- **Streamlit**
- **OpenAI API**
